**What would you like to be added**:
I'd like to increase log level for node filtration related events in TAS from 3 to 5.

**Why is this needed**:
3 is a commonly used log level to provide enhanced but not very low level logging information. It's more like DEBUG vs TRACE. With these node filtration events for large clusters we've seen hundreds of thousands of log lines emitted every minute. It can overwhelm some logging aggregators or trigger rate limiting. That's why I'd like to move these particular log events to a higher log level.
