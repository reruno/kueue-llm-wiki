## Description

**What happened**: Kueueviz backend fails to list cohorts due to missing rbac permissions.

**What you expected to happen**: Kueueviz frontend can display the cohorts.

**How to reproduce it (as minimally and precisely as possible)**:
1. Deploy kueueviz with default RBAC configuration
2. Access the kueueviz backend /ws/cohorts WebSocket endpoint
3. Observe errors: cohorts.kueue.x-k8s.io is forbidden: User "system:serviceaccount:kueue-system:kueueviz" cannot list resource "cohorts"
