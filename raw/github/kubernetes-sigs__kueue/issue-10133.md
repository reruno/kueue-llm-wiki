## Description

**What happened**:

Some integration tests are buggy because they pass LocalQueue to MakeAdmission rather than CLusterQueue as expected by the signature:
https://github.com/kaisoz/kueue/blob/aca13b5963a15ad2f1a10bf29e6e54b474c68493/pkg/util/testing/v1beta2/wrappers.go#L677

Example test which is passing LocalQueue name instead: https://github.com/kaisoz/kueue/blob/9e82591ef5522a249723b3644590b8f17ee83709/test/integration/singlecluster/controller/jobs/jobset/jobset_controller_test.go#L324

This sometimes works by "luck", but it makes the tests very hard to reason about.


**What you expected to happen**:

Use strong typing for the MakeAdmission to accept ClusterQueueReference rather than a string, so that all tests use it correctly.
