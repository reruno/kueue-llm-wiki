## Description

**What would you like to be added**:

Currently we have the [webook  ](https://github.com/kubernetes-sigs/kueue/blob/v0.15.3/pkg/controller/jobs/leaderworkerset/leaderworkerset_webhook.go#L243)
which does not allow to mutate the LWS. 

As a quick fix workaround if would be useful to investigate if the webhook could be disabled entirely on a cluster.

**Why is this needed**:

Some users would like to mutate the nodeSelector while the LWS running so that they can re-locate the LWS. It is ok if the LWS workloads are evicted and re-admitted.
