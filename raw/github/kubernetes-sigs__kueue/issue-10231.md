**What would you like to be cleaned**:

I would like to integrate and enforce the [lll](https://golangci-lint.run/docs/linters/configuration/#lll) (long line linter).

**Why is this needed**:

Long lines make code reviews more difficult, especially in side-by-side diffs, as they require horizontal scrolling and obscure the context of changes.
