## Description

**What would you like to be cleaned**:
We are currently maintaining a dependency on the https://github.com/kubernetes-sigs/cluster-inventory-api package to inject the secret reader plugin for e2e tests. Used here:
https://github.com/kubernetes-sigs/kueue/blob/9c0749e024fc787f63165c99053ec2c0bd8fd8ff/test/e2e/multikueue/e2e_test.go#L2175-L2178
and here:
https://github.com/kubernetes-sigs/kueue/blob/9c0749e024fc787f63165c99053ec2c0bd8fd8ff/test/e2e/multikueue/e2e_test.go#L2183-L2191

From what I can see, the plugins are already packaged into Docker images:
https://github.com/kubernetes-sigs/cluster-inventory-api/pkgs/container/cluster-inventory-api%2Fsecretreader/783588149?tag=0.1.0

But I think it's private for now. We could see whether it's going to be made public.

**Why is this needed**:
To simplify the end-to-end testing setup and reduce the maintenance overhead.
See: https://github.com/kubernetes-sigs/kueue/pull/10385, https://github.com/kubernetes-sigs/kueue/issues/10380, https://github.com/kubernetes-sigs/kueue/pull/10372
