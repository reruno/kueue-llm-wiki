## Description

**What would you like to be documented or improved**:

The `ProvisioningRequestConfigPodSetMergePolicy` field in the API reference does not currently list or describe the available values for this field. This may confuse users trying to use or configure this feature, as it is unclear what options are allowed.

The valid values, according to the API source code, are:

```go
// Valid enum values:
IdenticalPodTemplates
IdenticalWorkloadSchedulingRequirements
```
