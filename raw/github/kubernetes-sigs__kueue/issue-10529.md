## Description

<!-- Please only use this template for submitting enhancement requests -->

**What would you like to be added**:

I would like to add e2e tests, probably in the "customconfigs" suite which would detect the bug reported in https://github.com/kubernetes-sigs/kueue/issues/10495. In particular we want to check the preemptions are not blocked on the bug.

In particular they could guard against regressions related to any potential changes due to https://github.com/kubernetes-sigs/kueue/issues/10553

**Why is this needed**:

To ensure the problem does not return as a regression again. This bit of code seems quite sensitive yet not well tested.
