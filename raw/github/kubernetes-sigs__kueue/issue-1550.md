## Description

<!-- Please only use this template for submitting enhancement requests -->

**What would you like to be added**:
I would like to support other queues from the same cohort in the [`mechanism to stop a ClusterQueue`](https://github.com/kubernetes-sigs/kueue/tree/ed81667f0554345cfc5c956a28b8ad62564bcc81/keps/1284-cluster-queue-stop) (KEP-1284)

**Why is this needed**:
In static computing environments, we often restart nodes without autoscaling due to maintenance or hardware/os disruptions.
In such situations, I want to evict the jobs, borrowing resources from the same cohort.
