## Description

<!-- Please only use this template for submitting enhancement requests -->

**What would you like to be added**:

https://github.com/kubernetes-sigs/kueue/tree/main/keps/976-plain-pods#story-3

Queue a pod group using information about workers provided by a driver Pod.

**Why is this needed**:

In some frameworks, like Spark, worker Pods are created by a driver Pod. So we might not have the shape of a worker Pod until the driver runs.
