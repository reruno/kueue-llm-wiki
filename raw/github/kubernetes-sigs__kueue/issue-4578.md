## Description

<!-- Please only use this template for submitting clean up requests -->

**What would you like to be cleaned**:
Right now, kueue-viz frontend supports 2 env vars to specify the url of the backend:
- REACT_APP_WEBSOCKET_URL
- VITE_WEBSOCKET_URL

It should be better to have a framework agnostic variable like: `KUEUE_VIZ_BACKEND_WEBSOCKET_URL`

**Why is this needed**:
It should be better to have a framework agnostic variable like: `KUEUE_VIZ_BACKEND_WEBSOCKET_URL`
