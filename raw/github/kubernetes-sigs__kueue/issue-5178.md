## Description

<!-- Please only use this template for submitting clean up requests -->

**What would you like to be cleaned**:

We would like to benefit from a small Ray image, but without the need to maintain it inside Kueue.

**Why is this needed**:

The small Ray mini image reduces the build time of Kueue by around 6min, so we need it.

See more: https://github.com/kubernetes-sigs/kueue/pull/4917

At the same time maintaining the image comes with a cost, so we want to switch to the images hosted by the Ray project when available, see more here: https://github.com/ray-project/ray/issues/52375#issuecomment-2812872739
