## Description

**What would you like to be added**:

I would like to have two new dedicated issue templates for "Documentation" and "Flakes".

**Why is this needed**:

1. We often forget to add `/kind flake ` and `/kind documention` for such issues.
2. The template for bugs which is used for flakes contains unnecessary field so we need to remove each time
3. We can better guide for Documentation issues, like "which pages do you expect to be updated?"
