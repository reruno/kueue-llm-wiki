### Comment by [@skools-here](https://github.com/skools-here) — 2026-01-08T14:27:40Z

I’ve looked into this and the flakiness seems to come from aggressive shared timeouts in the multikueue integration test utilities. In CI, controller startup and cross-cluster reconciliation can exceed the current limits. I’m planning to propose a fix by increasing the common test timeouts to make the suite deterministic. I’ll open a PR shortly.
