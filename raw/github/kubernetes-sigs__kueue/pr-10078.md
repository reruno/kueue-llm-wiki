## Description

#### What type of PR is this?
/kind bug

#### What this PR does / why we need it:

Sets `DefaultTransform` to `TransformStripManagedFields()` on the
controller-runtime cache, stripping `metadata.managedFields` from all
objects before they enter the informer store.

Audit that outside of unit tests managedFields is [not used](https://github.com/search?q=repo%3Akubernetes-sigs%2Fkueue+managedFields+language%3AGo&type=code&l=Go). There was one other attempt to [strip these fields](https://github.com/kubernetes-sigs/kueue/pull/8342/changes) when comparing whether Node objects have changed 

#### Special notes for your reviewer:

#### Does this PR introduce a user-facing change?
```release-note
Strip managedFields from informer cache via DefaultTransform to reduce memory footprint on large clusters.
```
