#### What this PR does / why we need it:
This PR refactors unit tests to use a standardized featureGates map[featuregate.Feature]bool instead of individual boolean fields per feature gate.
This approach simplifies test setup, improves readability, and ensures consistency across the codebase when enabling or disabling feature gates.
