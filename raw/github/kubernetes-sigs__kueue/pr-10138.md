<blockquote>
<h2>v0.159.1</h2>
<p>The regression fixed in this release isn't new, but it's so subtle that we thought we'd release this sooner rather than later. For some time now, the minifier we use have stripped namespaced attributes in SVGs, which broke dynamic constructs using e.g. <a href="https://alpinejs.dev/directives/bind">AlpineJS' x-bind:</a> namespace (library used by Hugo's <a href="https://gohugo.io/">documentation site</a>).</p>
<p>To fix this, the upstream library has hadded a <code>keepNamespaces</code> slice option. It was not possible to find a default that would make all happy, so we opted for an option that at least would make AlpineJS sites work out of the box:</p>
<pre lang="toml"><code> [minify.tdewolff.svg]
      keepNamespaces = ['', 'x-bind']
</code></pre>
