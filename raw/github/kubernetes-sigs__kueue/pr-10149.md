#### What this PR does / why we need it:

Fixes `FairSharingPrioritizeNonBorrowing` in hierarchical cohorts by checking borrowing at every level of the cohort tree (CQ to root), not just at the CQ level. The check is per-flavor: a subtree borrowing on one flavor does not penalize workloads requesting a different flavor where the subtree has ample quota. The DRS struct now exposes which FlavorResources are borrowed, enabling callers to query borrowing against any set of flavors.
