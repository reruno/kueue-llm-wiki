Bumps [github.com/cert-manager/cert-manager](https://github.com/cert-manager/cert-manager) from 1.20.0 to 1.20.1.
<details>
<summary>Release notes</summary>
<p><em>Sourced from <a href="https://github.com/cert-manager/cert-manager/releases">github.com/cert-manager/cert-manager's releases</a>.</em></p>
<blockquote>
<p>v1.20.1 fixes an issue for OpenShift users that has to do with the finalizer RBAC, bumps gRPC to address a reported non-affecting vulnerability, and fixes a duplicate <code>parentRef</code> bug when both issuer config and annotations are present (Gateway API).</p>
<h3>Bug or Regression</h3>
<ul>
<li>Fixed duplicate <code>parentRef</code> bug when both issuer config and annotations are present. (<a href="https://redirect.github.com/cert-manager/cert-manager/issues/8658">#8658</a>, <a href="https://github.com/hjoshi123"><code>@​hjoshi123</code></a>)</li>
<li>Add missing issuer finalizer RBAC to the order controller to support owner references. This was preventing OpenShift users from being able to upgrade to v1.20.0. (<a href="https://redirect.github.com/cert-manager/cert-manager/issues/8655">#8655</a>, <a href="https://github.com/erikgb"><code>@​erikgb</code></a>)</li>
<li>Bump google.golang.org/grpc to fix vulnerability reported by scanners. This isn't a vulnerability that affects cert-manager, but we are bumping it because it is reported by scanners. (<a href="https://redirect.github.com/cert-manager/cert-manager/issues/8657">#8657</a>, <a href="https://github.com/erikgb"><code>@​erikgb</code></a>)</li>
</ul>
</blockquote>
