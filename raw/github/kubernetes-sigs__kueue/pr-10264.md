`2026-03-31T23:51:40.581620388Z stderr F 2026-03-31T23:51:40.581503957Z	LEVEL(-3)	scheduler	scheduler/tas_flavor.go:121	Constructing TAS snapshot	{"replica-role": "leader", "schedulingCycle": 2, "nodeLabels": {"cloud.provider.com/node-group":"tas-group"}, "levels": ["cloud.provider.com/topology-block", "cloud.provider.com/topology-rack", "kubernetes.io/hostname"], "nodeCount": 8}`

So I think we could include the flavorName into the context name there, by `WithValues("flavor", flavorName)`.
