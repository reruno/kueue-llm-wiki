## Pull request overview

This PR simplifies the repository’s Golangci-lint v2 configuration by enabling warnings for unused exclusion config and removing exclusion entries that no longer match any issues/paths in the repo, keeping the lint configuration lean and easier to maintain.

**Changes:**
- Enable `exclusions.warn-unused` for both `linters` and `formatters` to surface stale exclusion configuration.
- Remove unused `exclusions.presets`, `exclusions.rules`, and `exclusions.paths` entries that were producing “matched no issues / skipped 0 issues” warnings.
