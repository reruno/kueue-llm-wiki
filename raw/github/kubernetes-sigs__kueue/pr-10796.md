## Description

#### What type of PR is this?

/kind kep

#### What this PR does / why we need it:

Adds KEP-10765 for WorkloadPriorityClassDefaulting, a new feature gate that automatically assigns a WorkloadPriorityClass named default to workloads that don't specify one via the `kueue.x-k8s.io/priority-class` label.

In clusters where only a subset of workloads are managed by Kueue, this avoids conflating pod scheduling priority with workload queueing priority for jobs that omit the label, without requiring custom mutating webhooks or per-job labeling by users.

The design follows the same convention-based pattern as LocalQueueDefaulting.
