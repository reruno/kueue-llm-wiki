##### Include static code file
To embed an static code file into the a content page of the website, just add the the following code inside the page html file.
```
{{< include file="examples/<your-filepath>" lang="<your-code-language>" >}}
```
or without named parameters
```
{{< include "examples/<your-filepath>" "<your-code-language>" >}}
```

##### Include non-code static file
If you want include a non-code file,
```
{{<  include "<your-filepath>" >}}
```
##### Developer Notes
The first parameter of the `include` shortcode is a relative path to the `static` directory.  Note that you shouldn't use `{{% %}}` to call this shortcode, which could result in ill-formatted codeblocks. You should only use `{{<  >}}` to call this shortcode.
