#### What this PR does / why we need it:
As we discussed https://github.com/kubernetes-sigs/kueue/pull/1630#discussion_r1463848354, I added the `IntegrationOptions` to the `jobframework.Options`, and then I removed the `PodSelector` and `PodNamespaceSelector` from the `jobframework.Options`.

Also, I added a new function `CanSupportIntegration` to the `IntegrationCallbacks` interface so that we can intercept the termination of the manager if the integrations can not be supported in the cluster.

This PR is a prerequisite for #1630.
