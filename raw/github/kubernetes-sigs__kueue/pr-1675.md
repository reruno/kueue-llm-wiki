#### What this PR does / why we need it:
Dependabot is opening up PRs individually for k8s packages. We can actually group all the Kubernetes dependencies together to minimize the amount of PRs that we are getting for updates.

See https://github.com/cri-o/cri-o/pull/7707 for an example of what this syntax does.
