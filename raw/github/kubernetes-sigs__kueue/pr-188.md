#### What this PR does / why we need it:

The `guaranteed` and `ceiling` fields where named such for lack of better names.

`guranteed` is highly misleading, because without preemption and scheduler enforcement, we can't necessarily guarantee the quota. `min` seems less problematic. Then `max` makes more sense than `ceiling`.
