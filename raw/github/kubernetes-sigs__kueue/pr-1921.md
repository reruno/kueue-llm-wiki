#### What this PR does / why we need it:
We want to decrease the execution time of the e2e tests. In order to do so, I want to create a separate make target for the multikueue tests and remove multikueue tests from the existing test-e2e target (this PR) and run them in a separate job in the presubmits/periodic tests (separate PR as this is in a different repository).

Plus it fixes a typo in test/e2e/multikueue/suite_test.go
