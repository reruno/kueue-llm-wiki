## Description

#### What type of PR is this?

/kind cleanup

#### What this PR does / why we need it:

This PR reduces the time it takes to run the singlecluster e2e tests by removing the need to use long runtime containers in two tests 

The `pull-kueue-test-e2e-main-X-XX` test suite execution time went down **from ~80 seconds to ~65 seconds** (in all tested k8s versions):
* `Should unsuspend a job and set nodeSelectors`: from 9.505s to 4.182s
* `Should partially admit the Job if configured and not fully fits`: from 9.509s to 8.490s
* `Should unsuspend a job only after all checks are cleared`: from 10.803s to 5.967s


#### Which issue(s) this PR fixes:

Fixes #1449

NONE
