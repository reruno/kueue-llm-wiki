  IIUC the issue is that `output.String()` may return 1s rather than 0s as typically. However, I think it could still flake if `time.Now()` uses a different time than `output.String()` . I guess they would need to return different second, which is unlikely, but possible. 
  
  I think it is better to use `gomega.Eventually` to re-try if the times differ. And call `time.Now` and `output.String()` within the lambda.
  
  Alternatively, we could maybe use a fake clock, but I'm not sure how much refactoring this would be.
