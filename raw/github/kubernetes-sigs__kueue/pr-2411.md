  Maybe a better strategy is to calculate the "reason" in `GetTargets`, which already has all the information about the preemptor.
  
  We could have something like this in the preemption package:
  
  ```golang
  struct Target {
    *workload.Info
    Reason string
    Scope string // ClusterQueue/Cohort
  }
  ```
  
  Then `GetTargets` would return `[]Target`.
  
  Note that `e.assignment.Borrowing` is not enough to determine that it's fair sharing. This is because we have the feature `borrowWithinCohort` which can preempt while borrowing even if fair sharing is enabled. https://kueue.sigs.k8s.io/docs/concepts/cluster_queue/#preemption
  
  And that's all calculated in `GetTargets`.
