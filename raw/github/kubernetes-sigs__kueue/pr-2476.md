#### What this PR does / why we need it:

As discussed in https://github.com/kubernetes-sigs/kueue/pull/2377#issuecomment-2179242565 and https://kubernetes.slack.com/archives/CCK68P2Q2/p1718819420490889, we should start publishing Kueue images to `us-central1-docker.pkg.dev/k8s-staging-images/kueue`. 

Moreover, once https://github.com/kubernetes/k8s.io/pull/6910 is merged, we're going to have a staging repo for charts in `us-central1-docker.pkg.dev/k8s-staging-images/charts`.
