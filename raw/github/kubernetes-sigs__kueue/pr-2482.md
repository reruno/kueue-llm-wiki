#### What this PR does / why we need it:

Adds test case for undertested feature of flavor fungibility when `whenCanBorrow` and `whenCanPreempt` are both set to `TryNextFlavor`.

In the test case, it's not possible to borrow or preempt in the second flavor, but it's possible to preempt in the first flavor.
