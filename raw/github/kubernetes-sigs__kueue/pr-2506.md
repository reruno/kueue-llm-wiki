#### What this PR does / why we need it:
The async Google Analytics template from the Hugo project is no longer needed as has been deprecated - see https://github.com/gohugoio/hugo/issues/10899 -. This change is causing the error `no such template "_internal/google_analytics_async.html"` when running `make site-server` with the latest versions of Hugo.
