Hey @trasc @mimowo 
Apologies for the late response, I have made changes in above PR exactly as you suggested :

Actually my usecase is particularly related to air-gapped environment where the ImageContentSourcePolicy(ICSP) and ClusterImageSet(CIS) are adjusted such as the images required must be referred to digest rather than tag.
The disconnected cluster pulls images from a specific set of mirrors instead of their original registry location. When ICSP is applied, the cluster uses image digest instead of tag to ensure consistency and immutability of images across different mirrors.

In case of running e2e tests in such disconnected environment, for this PR I considered 3 scenarios :
1. It will consider `gcr.io/k8s-staging-perf-tests/sleep:v0.1.0` as default image for running E2E tests 
2. Incase of providing different image, it can be provided using `E2E_TEST_IMAGE` environment variable (I guess this might help in preserving kind cluster e2e-test behaviour - https://github.com/kubernetes-sigs/kueue/pull/2570#discussion_r1690951585)
2. To use image digest version of same default image (I need your opinion on which method to select) :

	i)   First way as suggested by @mimowo - https://github.com/kubernetes-sigs/kueue/pull/2570#issuecomment-2244595066, used separate `E2E_TEST_IMAGE_DIGEST` environment variable to specify digest to be used with the default e2e-test-image.. (Used this method)
        ii)  Other way is to use FetchImageDigest() method in util/e2e.go  to fetch image digest of an existing default image similar to below mentioned code snippet, but it requires docker-cli to be installed as a pre-requisite (So skipped)


```

func FetchImageDigest(image string) (string, error) {
	client, err := dockerClient.NewClientWithOpts(dockerClient.FromEnv)
	if err != nil {
	      return "", err
	}
	imageInspect, _, err := client.ImageInspectWithR(context.Background(), image)
	if err != nil {
	      return "", err
	}
	if len(imageInspect.RepoDigests) > 0 {
	      digest := strings.Split(imageInspect.RepoDigests[0], "@")[1]
	      return digest, nil
	}
	return "", fmt.Errorf("digest not found for image %s", image)
}

```
