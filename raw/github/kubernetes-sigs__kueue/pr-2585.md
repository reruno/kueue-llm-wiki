<h2>Kueue v0.7.1</h2>
<p>Changes since <code>v0.7.0</code>:</p>
<h3>Feature</h3>
<ul>
<li>Improved logging for scheduling and preemption in levels 4 and 5 (<a href="https://redirect.github.com/kubernetes-sigs/kueue/issues/2510">#2510</a>, <a href="https://github.com/gabesaba"><code>@​gabesaba</code></a>, <a href="https://github.com/alculquicondor"><code>@​alculquicondor</code></a>)</li>
<li>MultiKueue: Remove remote objects synchronously when the worker cluster is reachable. (<a href="https://redirect.github.com/kubernetes-sigs/kueue/issues/2360">#2360</a>, <a href="https://github.com/trasc"><code>@​trasc</code></a>)</li>
</ul>
<h3>Bug or Regression</h3>
<ul>
<li>Fix check that prevents preemptions when a workload requests 0 for a resource that is at nominal or over it. (<a href="https://redirect.github.com/kubernetes-sigs/kueue/issues/2524">#2524</a>, <a href="https://github.com/mbobrovskyi"><code>@​mbobrovskyi</code></a>, <a href="https://github.com/alculquicondor"><code>@​alculquicondor</code></a>)</li>
<li>Fix for the scenario when a workload doesn't match some resource flavors due to affinity or taints
could cause the workload to be continuously retried. (<a href="https://redirect.github.com/kubernetes-sigs/kueue/issues/2440">#2440</a>, <a href="https://github.com/KunWuLuan"><code>@​KunWuLuan</code></a>)</li>
<li>Fix missing fairSharingStatus in ClusterQueue (<a href="https://redirect.github.com/kubernetes-sigs/kueue/issues/2432">#2432</a>, <a href="https://github.com/mbobrovskyi"><code>@​mbobrovskyi</code></a>)</li>
<li>Fix missing metric cluster_queue_status. (<a href="https://redirect.github.com/kubernetes-sigs/kueue/issues/2475">#2475</a>, <a href="https://github.com/mbobrovskyi"><code>@​mbobrovskyi</code></a>)</li>
<li>Fix panic that could occur when a ClusterQueue is deleted while Kueue was updating the ClusterQueue status. (<a href="https://redirect.github.com/kubernetes-sigs/kueue/issues/2464">#2464</a>, <a href="https://github.com/mbobrovskyi"><code>@​mbobrovskyi</code></a>)</li>
<li>Fix panic when there is not enough quota to assign flavors to a Workload in the cohort, when FairSharing is enabled. (<a href="https://redirect.github.com/kubernetes-sigs/kueue/issues/2449">#2449</a>, <a href="https://github.com/mbobrovskyi"><code>@​mbobrovskyi</code></a>)</li>
<li>Fix performance issue in logging when processing LocalQueues. (<a href="https://redirect.github.com/kubernetes-sigs/kueue/issues/2492">#2492</a>, <a href="https://github.com/alexandear"><code>@​alexandear</code></a>)</li>
<li>Fix race condition on delete workload from queue manager. (<a href="https://redirect.github.com/kubernetes-sigs/kueue/issues/2465">#2465</a>, <a href="https://github.com/mbobrovskyi"><code>@​mbobrovskyi</code></a>)</li>
<li>MultiKueue: Do not reject a JobSet if the corresponding cluster queue doesn't exist (<a href="https://redirect.github.com/kubernetes-sigs/kueue/issues/2442">#2442</a>, <a href="https://github.com/vladikkuzn"><code>@​vladikkuzn</code></a>)</li>
<li>MultiKueue: Skip garbage collection for disconnected clients which could occasionally result in panic. (<a href="https://redirect.github.com/kubernetes-sigs/kueue/issues/2370">#2370</a>, <a href="https://github.com/trasc"><code>@​trasc</code></a>)</li>
<li>Show weightedShare in ClusterQueue status.fairSharing even if the value is zero (<a href="https://redirect.github.com/kubernetes-sigs/kueue/issues/2522">#2522</a>, <a href="https://github.com/alculquicondor"><code>@​alculquicondor</code></a>)</li>
<li>Skip duplicate Tolerations when an admission check introduces a toleration that the job also set. (<a href="https://redirect.github.com/kubernetes-sigs/kueue/issues/2499">#2499</a>, <a href="https://github.com/trasc"><code>@​trasc</code></a>)</li>
</ul>
