#### Special notes for your reviewer:
```
Additional environment variables

Each container started via kjobctl generated yamls will get the following environment variables:

USER_ID - Executing user id ($USER from the machine where kjobctl was executed).
TASK_ID - Unique identifier of the task (not pod) - most likely $USER_ID+$TIMESTAMP+$TASK_NAME
TASK_NAME - namespace +”_”+ name.
PROFILE - namespace +  “_” + profile that was used to create the task.
TIMESTAMP - timestamp when the task was created.
```
