#### What this PR does / why we need it:
The primary motivation is to resolve the considerations of merging dependabot PRs.
The current dependabot often enforces us to consider the order of merging dependabot PRs like 

- https://github.com/kubernetes-sigs/kueue/pull/2849
- https://github.com/kubernetes-sigs/kueue/pull/2846

#2849 failed due to merge #2846 first.

Since this PR, we group codepending directories during automated dependabot go module update:

- Kueue
      - "/"
      - "/hack/internal/tools"
      - "/site"
- KJobCtl
      - "/cmd/experimental/kjobctl"
      - "/cmd/experimental/kjobctl/hack/tools"
