#### Special notes for your reviewer:
We have warnings:

```
 => WARN: InvalidDefaultArgInFrom: Default value for ARG ${BUILDER_IMAGE} results in empty or invalid base image name (line 4)                                                0.0s
 => WARN: InvalidDefaultArgInFrom: Default value for ARG ${BASE_IMAGE} results in empty or invalid base image name (line 17)                                                  0.0s
```

on execute `make kind-image-build`.

As mentioned on docs [here](https://docs.docker.com/reference/build-checks/invalid-default-arg-in-from/) we should set default value for args that is used on the `FROM`.
