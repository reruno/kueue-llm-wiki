<blockquote>
<h2>Kueue v0.8.1</h2>
<p>Changes since <code>v0.8.0</code>:</p>
<h3>Feature</h3>
<ul>
<li>Add gauge metric admission_cycle_preemption_skips that reports the number of Workloads in a ClusterQueue
that got preemptions candidates, but had to be skipped in the last cycle. (<a href="https://redirect.github.com/kubernetes-sigs/kueue/issues/2942">#2942</a>, <a href="https://github.com/alculquicondor"><code>@​alculquicondor</code></a>)</li>
<li>Publish images via artifact registry (<a href="https://redirect.github.com/kubernetes-sigs/kueue/issues/2832">#2832</a>, <a href="https://github.com/alculquicondor"><code>@​alculquicondor</code></a>)</li>
</ul>
<h3>Bug or Regression</h3>
<ul>
<li>CLI: Support <code>-</code> and <code>.</code> in the resource flavor name on <code>create cq</code> (<a href="https://redirect.github.com/kubernetes-sigs/kueue/issues/2706">#2706</a>, <a href="https://github.com/trasc"><code>@​trasc</code></a>)</li>
<li>Detect and enable support for job CRDs installed after Kueue starts. (<a href="https://redirect.github.com/kubernetes-sigs/kueue/issues/2991">#2991</a>, <a href="https://github.com/ChristianZaccaria"><code>@​ChristianZaccaria</code></a>)</li>
<li>Fix over-admission after deleting resources from borrowing ClusterQueue. (<a href="https://redirect.github.com/kubernetes-sigs/kueue/issues/2879">#2879</a>, <a href="https://github.com/mbobrovskyi"><code>@​mbobrovskyi</code></a>)</li>
<li>Fix support for kuberay 1.2.x (<a href="https://redirect.github.com/kubernetes-sigs/kueue/issues/2983">#2983</a>, <a href="https://github.com/mbobrovskyi"><code>@​mbobrovskyi</code></a>)</li>
<li>Helm: Fix a bug for &quot;unclosed action error&quot;. (<a href="https://redirect.github.com/kubernetes-sigs/kueue/issues/2688">#2688</a>, <a href="https://github.com/mbobrovskyi"><code>@​mbobrovskyi</code></a>)</li>
<li>Prevent infinite preemption loop when PrioritySortingWithinCohort=false
is used together with borrowWithinCohort. (<a href="https://redirect.github.com/kubernetes-sigs/kueue/issues/2831">#2831</a>, <a href="https://github.com/mimowo"><code>@​mimowo</code></a>)</li>
<li>Support for helm charts in the us-central1-docker.pkg.dev/k8s-staging-images/charts repository (<a href="https://redirect.github.com/kubernetes-sigs/kueue/issues/2834">#2834</a>, <a href="https://github.com/IrvingMg"><code>@​IrvingMg</code></a>)</li>
<li>Update Flavor selection logic to prefer Flavors which allow reclamation of lent nominal quota, over Flavors which require preempting workloads within the ClusterQueue. This matches the behavior in the single Flavor case. (<a href="https://redirect.github.com/kubernetes-sigs/kueue/issues/2829">#2829</a>, <a href="https://github.com/gabesaba"><code>@​gabesaba</code></a>)</li>
</ul>
</blockquote>
