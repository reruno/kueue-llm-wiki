#### Special notes for your reviewer:
In tests, we set the old workload at the first place, but in `ValidateWorkloadUpdate` the first parameter is newObj. Mismatch here. 
https://github.com/kubernetes-sigs/kueue/blob/21091c614ea5261664274418c676cf71eee01f0c/apis/kueue/v1alpha1/workload_webhook_test.go#L145-L148

`func(oldObj, newObj)` seems more native than `func(newObj, oldObj)`.
