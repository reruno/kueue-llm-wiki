#### What this PR does / why we need it:

Accumulate the total duration that a workload has been in the 'Admitted' state across previous Admit/Evict cycles, and utilize this when determining if the maximum execution time has been surpassed.
This counter is set to 0 when the workload gets deactivated due to maximum execution time exceeded.
