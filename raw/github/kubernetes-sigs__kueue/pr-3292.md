## Description

#### What type of PR is this?

/kind feature

#### What this PR does / why we need it:

To include capacity from non-TAS pods. Mainly DaemonSets, but not only.

#### Which issue(s) this PR fixes:

Part of #2724 

#### Special notes for your reviewer:

I have initially done indexing of all  non-TAS, scheduled and non-terminal pods, but it is more code, so I decided for now to do minimal change and filter the terminal pods when accounting for capacity.
