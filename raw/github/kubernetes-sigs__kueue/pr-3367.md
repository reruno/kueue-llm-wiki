
  Actually, does it make sense to even check `time.Since(admittedAt) < podsReadyTimeout` this way? If the g.Expect fails it will be retried by `gomega.Eventually`. 
  
  So, IIUC we can just sleep(podsReadyTimeout) and wrap the gomega.Eventually around Expect for WorkloadEvicted. Note that if we do `sleep(podsReadyTimeout)` the Eventually the test will be sligtly faster on average as without the sleep the Eventually will backoff with exponential delay so it is unlikely to be checked after 2s (it will be checked after 2s + the accumulated backoff).
  
  Then, verify `g.Expect(ptr.Deref(prodWl1.Status.RequeueState,` outside of the Eventually.
