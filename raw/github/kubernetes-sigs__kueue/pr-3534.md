#### What this PR does / why we need it:
`kjobctl describe slurm `currently prints out the content of the executed script, but not the name of the source file of the script.it makes it easier for the user to identify the work the job does without reading the script itself
use cases in scripts using the kjob - pulling the name - also for identification purposes
