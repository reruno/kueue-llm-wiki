#### What this PR does / why we need it:

The `-v` option in conditional expressions was introduced to bash in versions later than v3. Therefore, those versions that don't support it fail to run e2e tests returning the error `conditional binary operator expected`.

To avoid the above error, this PR replaces the condition to check if a variable is set.
