<h2>v0.139.0</h2>
<p>This release is mostly about removing code that has been deprecated for a long time. This has been shown as an <code>ERROR</code> in the build log and failed the build for at least the last 6 minor Hugo versions, in most cases much longer.</p>
<p>But this is also a full dependency refresh, and there are also some new stuff. For one, we added a <code>-O</code> flag to <code>hugo server</code> to open up the site in your browser after start. This is now my (<a href="https://github.com/bep"><code>@​bep</code></a>) shortcut to start the server:</p>
<pre lang="zsh"><code>function h() {
  hugo server  -F -O -N -D
}
</code></pre>
