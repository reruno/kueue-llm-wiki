### Review by [@mimowo](https://github.com/mimowo) (commented) — 2025-04-10T14:16:51Z

I'm not entirely sure this works ok, I think we need e2e test. For example:
1. create a Deployment with queue-name and 2 suspended pods because there is not quota
2. remove queue-name from deployment
Will this replace the Pods which are gated? If the Pods are not replaced, then I think it could be a problem if the quota is released, because the Pods from (1.) will still have queue-name

So, I think it is more complex than looks on the surface. Please test manually at least.
