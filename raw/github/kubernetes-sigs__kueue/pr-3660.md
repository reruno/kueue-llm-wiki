#### What this PR does / why we need it:
Currently, we re declare the error type in the clusterqueue webhook integration testings and reuse those in most of webhook integration test cases.

https://github.com/kubernetes-sigs/kueue/blob/31c421356d8decd616fd37ac063aa3cf1b82e1e8/test/integration/webhook/core/clusterqueue_test.go#L38-L42

However, we already have dedicated error matcher. So, I remove the redundant error type checkings and reuse the testing utils.

https://github.com/kubernetes-sigs/kueue/blob/31c421356d8decd616fd37ac063aa3cf1b82e1e8/pkg/util/testing/error_matchers.go
