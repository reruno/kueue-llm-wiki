<h2>Features</h2>
<h3>Play or pause the timer programmatically</h3>
<p>By default, when the notification is hovered or the window loses focus, the timer for dismissing the notification is paused. There are many other situations where you might want to pause the timer as well. For instance, consider wanting to toggle the notification timer based on the document's visibility. This wasn't possible to do previously, but with the new API, it's a breeze.</p>
<pre lang="tsx"><code>document.addEventListener(&quot;visibilitychange&quot;, () =&gt; {
  if (document.visibilityState === &quot;visible&quot;) {
    toast.play({ id: &quot;123&quot; });
  } else {
    toast.pause({ id: &quot;123&quot; });
  }
});
</code></pre>
<p>More usages:</p>
<ul>
<li>Play/pause all toasts</li>
</ul>
<pre lang="tsx"><code>toast.play()
toast.pause()
</code></pre>
<ul>
<li>Play/pause all toasts for a given container</li>
</ul>
<pre lang="tsx"><code>toast.play({ containerId: &quot;123&quot; })
toast.pause({ containerId: &quot;123&quot; })
</code></pre>
<ul>
<li>Play/pause toast that has a given id regardless the container</li>
</ul>
<pre lang="tsx"><code>toast.play({ id: &quot;123&quot; })
toast.pause({ id: &quot;123&quot; })
</code></pre>
<ul>
<li>Play/pause toast that has a given id for a specific container</li>
</ul>
<pre lang="tsx"><code>toast.play({ id: &quot;123&quot;, containerId: &quot;12&quot; })
toast.pause({ id: &quot;123&quot;, containerId: &quot;12&quot; })
</code></pre>
