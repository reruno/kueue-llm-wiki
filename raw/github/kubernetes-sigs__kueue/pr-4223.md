#### What this PR does / why we need it:
As I described in https://github.com/kubernetes-sigs/kueue/pull/4222#issuecomment-2650505614, the cloudbuild or DinD does not support the multi-architecture container buildings by `docker buildx build --platform A,B,C`.

If we want to perform concurrently image buildings, we need to build a single arch image, and then amend those image manifests into one.
