#### What this PR does / why we need it:
For the ease of local debugging and testing in IDE, where tests are not always run with Makefile.
Default the ProjectDirBase variable by walking up the tree of folders until we encounter a folder with Makefile, only as a fallback when PROJECT_DIR is not specified.
