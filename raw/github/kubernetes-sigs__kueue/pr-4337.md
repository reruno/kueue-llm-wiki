#### Special notes for your reviewer:

There are three changes of note that I'm uncertain about:
1. I've added a client to the internal clusterQueue representation. There was one already available as part of the TAS cache.
2. I've added a reference to the lq object as part of the internal lq representation. The LocalQueue (or at least its metadata) is required when checking the truth value of the label selectors
3. I've added `ctx` to many functions so that it can trickle down to the ShouldReportLocalMetricsFunction
