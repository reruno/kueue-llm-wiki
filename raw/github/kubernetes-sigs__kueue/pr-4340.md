#### What this PR does / why we need it:
When using RayJob with `SubmissionMode = K8sJobMode`.
The Submitter PodSet does not contain TopologyRequest thus RayJob running with this submission mode can't be used with TAS.
To fix it we need to add Topology Request to the Submitter PodSet.
