## Description

This is because some components like helm or argocd will add its own labels to owned objects
for management, in this situation, when we use them to deploy resourceFlavor, unexpected labels
will be added to resourceFlavor which will lead to pods pending for corresponding jobs will be added
these labels to their node selector automatically
