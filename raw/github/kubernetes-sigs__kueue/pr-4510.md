#### What this PR does / why we need it:

#### Which issue(s) this PR fixes:

Follow up to https://github.com/kubernetes-sigs/kueue/pull/4418 to:
- make the code easier to read by unifying the concept of Usage for admitted workload (Quota and TAS usage)
- make the code faster by recording the TASUsage so that it does not need to be recomputed
- rename usage() to admittedUsage() to minimize the confusion with Usage as discussed in https://github.com/kubernetes-sigs/kueue/pull/4503#discussion_r1981349239
