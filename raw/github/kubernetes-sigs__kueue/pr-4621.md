### Comment by [@mimowo](https://github.com/mimowo) — 2025-03-17T07:27:50Z

> How do these args change the exit strategy for the images?

Yeah, the differences should be documented with comments. Basically the names correspond to different commands used in the agnhost names. I don't think the names are also best at expressing the intention. Eg. the "netexec" command is much more versatile than waiting for deletion. I'm happy if someone revisits this.

```
	BehaviorWaitForDeletion           = []string{"netexec"}	
	BehaviorWaitForDeletionFailOnExit = []string{"pause"}
	BehaviorExitFast                  = []string{"entrypoint-tester"}
```
