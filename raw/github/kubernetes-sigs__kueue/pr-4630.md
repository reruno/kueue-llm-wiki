#### What this PR does / why we need it:

I introduced the `TypedControllerManagedBy` instead of `ControllerManagedBy` in the workload controller.
This allows us to use the `TypedPredicate` rather than `Predicate`, which allows us to avoid typed inference (`obj, ok := e.Object.(*kueue.Workload)`) everywhere.
