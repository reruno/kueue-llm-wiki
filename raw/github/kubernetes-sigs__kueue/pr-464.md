## Description

Before we will have a default name  for unset podSet name, it's auto generated via kubebuilder. But when we have two name unset podSets, they will both have the  name, it violates the API validation of non-repeated names. So remove the annotation in the API but add the default name via webhook muatation only when we have one podSet and its name is empty
