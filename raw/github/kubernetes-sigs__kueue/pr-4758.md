#### What this PR does / why we need it:
On a failed build, Cypress generates screenshots:

```
cmd/experimental/kueue-viz/frontend/cypress/screenshots
└── app.cy.js
    ├── Kueue Dashboard -- should navigate to cluster-queue and verify local queues (failed).png
    ├── Kueue Dashboard -- should navigate to resource-flavors and verify table content (failed).png
    └── Kueue Dashboard -- should verify cohort link and navigate to cohorts page (failed).png
```

We should ignore this directory to prevent accidentally adding these files to git.
