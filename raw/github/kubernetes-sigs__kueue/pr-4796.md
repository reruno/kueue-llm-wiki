#### What this PR does / why we need it:
Right now we load all the images to all the nodes in the single cluster e2e tests.
However there is no need to spend precious seconds (multiplied by number of images and their size) to do that for `control-plane` node.
This is especially true when we talk about e.g. Ray-project image that has 2.2GB.
