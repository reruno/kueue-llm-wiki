  In the `LeaderWorkerSet` integration, we set default values via the `LeaderWorkerSetPodReconciler` after `Pod` creation. 
  
  https://github.com/kubernetes-sigs/kueue/blob/13574fcc4dc829cc86c36e3e15fc4d68739f074f/pkg/controller/jobs/leaderworkerset/leaderworkerset_pod_reconciler.go#L127-L131

  We should ensure that we wait for the `LeaderWorkerSetPodReconciler` to apply the default values before setting the `ManagedByKueueLabel`.
