#### What this PR does / why we need it:
New version of Kubeflow training-Operator(v1.9.1) does not place image at the deployment.
That was caught by the dependabot run: https://github.com/kubernetes-sigs/kueue/pull/4836
It could be consider a bug on the Kubeflow side or we could adjust.

This change takes the image version from the kustomization.yaml directly.
