#### What this PR does / why we need it:

We need smaller version of ray-project image to speed up Ray related tests.
For reference ray-project compressed [image](https://hub.docker.com/layers/rayproject/ray/2.41.0-py39/images/sha256-d57d976a11ba6ef2e2eb6184e8c5523db4b47d1b86b1036d255359824c8d40a0) size 873,79MB and uncompressed is 2.21GB, while ray-project-mini is 144.19 MB compressed and 415MB uncompressed.

The `ray-project-mini` image is being build before the e2e tests that use it.
The approach is temporary solution, the final goal is to have in Kuberay registry or similar.

Example time comparison for `pull-kueue-test-e2e-multikueue-main`
