#### What this PR does / why we need it:
We need smaller version of ray-project image to speed up Ray related tests.
RayJobs requieres additional logic that had to be mocked in order to allow RayJobs to start and finish in e2e tests.
RayCluster would work without extra logic.
However RayJob calls couple ray endpoints e.g. to ask for ray version, to check job logs after it has finished.

In another PR this image will be used to run in e2e tests.
