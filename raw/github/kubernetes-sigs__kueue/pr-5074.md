## Description

#### What type of PR is this?

/kind bug

#### What this PR does / why we need it:

Add an extra label to the service for metrics, for the ServiceMonitor to select that Service only, and not any other Service in the same namespace matching the common labels.

#### Which issue(s) this PR fixes:

Fixes #5072
