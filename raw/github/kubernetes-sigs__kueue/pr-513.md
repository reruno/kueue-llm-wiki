## Description

#### What type of PR is this?

/kind feature

#### What this PR does / why we need it:

Currently there is no easy easy way to increase the default QPS/Burst settings of 20/30 (imposed by controller-runtime). This PR adds QPS/Burst settings to the config file and bumps the default values to 25/35. Larger deployments may require increasing these values even further.

```

```
