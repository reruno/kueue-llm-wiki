<h2>v2.7.0</h2>
<h2>2.7.0</h2>
<h3>Features</h3>
<ul>

<li>Color aliases for custom color support (<a href="https://github-redirect.dependabot.com/onsi/ginkgo/issues/1101">#1101</a>) [49fab7a]</li>
</ul>
<h3>Fixes</h3>
<ul>
<li>correctly ensure deterministic spec order, even if specs are generated by iterating over a map [89dda20]</li>
<li>Fix a bug where timedout specs were not correctly treated as failures when determining whether or not to run AfterAlls in an Ordered container.</li>
<li>Ensure go test coverprofile outputs to the expected location (<a href="https://github-redirect.dependabot.com/onsi/ginkgo/issues/1105">#1105</a>) [b0bd77b]</li>
</ul>
