#### What this PR does / why we need it:

Currently TAS allows to request a topology for the whole podset which sometimes is not enough. This PR adds a possibility to specify "chunk topology" in addition to a usual podset topology. PodSet Chunk is a subset of podset. Podset divides into equally sized chunks.

User can specify required/preferred topology for the whole podset as well as a required lower-level topology for each podset chunk.
