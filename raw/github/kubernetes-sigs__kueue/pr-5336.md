#### What this PR does / why we need it:
Adds namespace filtering functionality to KueueViz backend and frontend. Implements a new /namespaces endpoint that returns namespaces containing LocalQueues, and adds namespace query parameter support to workloads and dashboard endpoints. Includes corresponding UI components with dropdown filters in both Dashboard and Workloads pages.
