#### What this PR does / why we need it:

This PR adds Slice fields to Workload and defines annotations to set those values, but they are not used in this PR. They'll be used in https://github.com/kubernetes-sigs/kueue/pull/5353.

It also adds a defaulting mechanism for JobSet that sets SliceSize to the value of parallelism if it's not defined.

It's a preparatory PR for: https://github.com/kubernetes-sigs/kueue/pull/5353
