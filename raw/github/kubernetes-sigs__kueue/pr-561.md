#### What this PR does / why we need it:

It is needed for the Kueue integration with other frameworks. The first integration we will need it for is with the MPI operator framework. The annotation will be added by a webhook to the child job created by a framework (the Launcher job in case of the MPIJob). A job with this annotation will not create a Kueue workload, but reuse the workload created by its parent job (the MPIJob in the case of the integration with MPIJob).
