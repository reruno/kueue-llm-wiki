## Description

In case the kueue image is built without using the
`image-build` from the Makefile, the following arg
would be evaluated as `CGO_ENABLED=`, without a value,
causing the kueue-controller-manager to fail with
`exec /manager: no such file or directory`. This commit
fixes the issue by ensuring that argument has a default value.
