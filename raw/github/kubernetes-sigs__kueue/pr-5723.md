#### What this PR does / why we need it:
Current implementation of TAS slices works only if slice size evenly divides number of pods in PodSet. We agreed that it is a good idea to allow for flexibility here. If sliceSize does not evenly divide the number of pods then the last slice will be a partial slice containing lower number of pods than sliceSize. This PR adjusts calculations to account for that.
