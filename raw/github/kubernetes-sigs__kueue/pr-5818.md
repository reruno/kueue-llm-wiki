#### What this PR does / why we need it:

The GroupKindConcurrency in the Kueue Config must be propagated to each controller in the following:

https://github.com/kubernetes-sigs/controller-runtime/blob/4c2e9cec2954e6d5a9522c65b2e22f24acb750b5/pkg/config/controller.go#L35-L45

However, some controllers use the `WatchesRawSource` instead of `For` for registrations. So, in the case of `WatchesRawSource`, we need to manually specify the concurrency propagated from Kueue Config.
