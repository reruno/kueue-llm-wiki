To allow calling `IMAGE_BUILD_EXTRA_OPTS="-o type=docker,dest=bin/kueue-image-dev.tar" make image-local-build`. This does not work currently, because then it calls `make image-build PUSH= IMAGE_BUILD_EXTRA_OPTS=-o type=docker,dest=bin/kueue-image-dev.tar`, resulting in 

```
docker buildx build \
...
	 \
	-o \  <- split on the first space
	./
```
