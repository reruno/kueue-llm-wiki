#### What this PR does / why we need it:
Node failure controller removes the `NodeToReplace` annotation if the node recovers or reappears.

This is to handle the case where a node recovers (after the 30s `NodeFailureDelay` but before the 3m `RecoveryTimeout`). This is especially important if no replacement is possible for this node.
