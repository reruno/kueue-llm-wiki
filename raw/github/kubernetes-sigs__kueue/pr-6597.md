#### What this PR does / why we need it:

This PR adds a alpha native integration of [Kubeflow Trainer](https://github.com/kubeflow/trainer/releases/tag/v2.0.0-rc.0) to Kueue. 

#### Why is this needed:

It will help provide early feedback and make sure there are no roadblocks for the full integration in #3884 

#### Which issue(s) this PR fixes:
Fixes #5719 

#### Special notes for your reviewer:

* The current TrainJob API does not support setting labels and annotations per job. Users can set the labels directly on runtime template to configure TAS-specific labels for Kueue.
* The TrainJob controller admission webhook rejects modifications to `podSpecOverrides` if any job is active. This conflicts with Kueue’s behavior, which attempts to restore previous `PodSet` config when stopping a job. For this alpha integration, I overwrite `podSpecOverrides` at each job start. A long-term solution should be discussed for #3884.
* Added a few TODO comments to help the reviewer which will be cleaned up later once they’re not needed

Not sure if just for me but while testing i encountered a crash in the Kueue controller, but got it sorted with min tweaks.. otherwise it worked seamlesly 🚀  :

When deploying with `config/alpha-enabled` (which uses `--feature-gates=AllAlpha=true`), the controller crashes with a nil pointer dereference resulting in CrashLoopBackOff:
```
panic: runtime error: invalid memory address or nil pointer dereference
at sigs.k8s.io/kueue/pkg/util/admissionfairsharing.CalculateEntryPenalty
```
IIUC AllAlpha=true flag enables the `AdmissionFairSharing` feature, but no admissionFairSharing configuration is provided in the default config: 
- AdmissionFairSharing feature = enabled (due to AllAlpha=true)
- but as by default the AdmissionFairSharing config was commented out by default in [kueue-manager-config](https://github.com/kubernetes-sigs/kueue/blob/0666f2ceb937a8f2d9198e381f9dd5cd19445d1a/charts/kueue/values.yaml#L164) configmap in kueue-system ns
- cfg.AdmissionFairSharing = nil (no config provided)
- Scheduler calls [CalculateEntryPenalty(requests, nil)](https://github.com/kubernetes-sigs/kueue/blob/0666f2ceb937a8f2d9198e381f9dd5cd19445d1a/pkg/util/admissionfairsharing/admission_fair_sharing.go#L47)
- Crash at line 49: afs.UsageSamplingInterval.Seconds()

Results in : 
- Controller enters CrashLoopBackOff
- TrainJob workloads cannot be admitted

What worked for me : 

- Added a nil check in pkg/util/admissionfairsharing/admission_fair_sharing.go:
```
func CalculateEntryPenalty(totalRequests corev1.ResourceList, afs *config.AdmissionFairSharing) corev1.ResourceList {
	// Return empty penalty if no admission fair sharing config is provided
	if afs == nil {
		return corev1.ResourceList{}
	}
	alpha := CalculateAlphaRate(
		afs.UsageSamplingInterval.Seconds(),
		afs.UsageHalfLifeTime.Seconds(),
	)
	return resource.MulByFloat(totalRequests, alpha)
}
```
- or uncomment [kueue-manager-config](https://github.com/kubernetes-sigs/kueue/blob/0666f2ceb937a8f2d9198e381f9dd5cd19445d1a/charts/kueue/values.yaml#L164)'s AdmissionFairSharing config
