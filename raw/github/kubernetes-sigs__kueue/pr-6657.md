#### What this PR does / why we need it:
This PR replaces `t.Context()` usage in unit tests with `utiltesting.ContextWithLog(t)`.  
The new helper provides a logger bound to the test context, which enables better visibility during test runs when `TEST_LOG_LEVEL=8` is set.
