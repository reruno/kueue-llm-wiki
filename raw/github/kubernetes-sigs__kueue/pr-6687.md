  **About the name `invalid request with validate on update`**
  
  IIUC, “with validate” is meant to exercise the `jobWithValidation, ok := newJob.(JobWithCustomValidation); ok { ... }` path.   1. Prefer a dedicated unit test that uses `KubeflowJob` as the backing object to actually cover the validation branch, or
  2. Change the test object in this case to `KubeflowJob`.
