## Description

This change introduces a generic MultiKueue adapter to support external, custom Job-like workloads. This allows Kueue to manage resources for CRDs that follow a Job-like pattern but are not part of the core supported set.

Key changes include:
- A new `generic_multikueue_adapter` that can be configured to watch and manage custom resources.
- A new feature gate, `MultiKueueGenericJobAdapter`, to enable this functionality.
This work implements the design outlined in KEP-2349.
