#### What this PR does / why we need it:
This PR extends the MultiKueue API to support worker cluster authentication using the ClusterProfile API, as an alternative to kubeconfig.

This change introduces a more flexible and secure method for managing credentials by leveraging the credential provider plugin mechanism described in [KEP-5339](https://github.com/kubernetes/enhancements/blob/master/keps/sig-multicluster/5339-clusterprofile-plugin-credentials/README.md). It also introduces a new ClusterProfile controller responsible for syncing ClusterProfile objects to MultiKueueCluster objects, simplifying worker cluster management. #6714
