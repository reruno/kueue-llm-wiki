#### What this PR does / why we need it:
Deflakes an integration test. The name of the test was misleading as it is about a node becoming available (not recovering). I checked the history of revisions of this test and it never tested the recovery of the node. I'm renaming it to avoid confusion. Also, it can only work if the FailFast flag is set to false. So, I'm changing the flag to false for the duration of the test. 

Tested by running this test in a loop for an ~hour.
