## Description

#### What type of PR is this?
/kind bug


#### What this PR does / why we need it:
resource.Quantity can only represent down to 1 nano (10^-9). See https://github.com/kubernetes-sigs/kueue/issues/4247#issuecomment-3312129775

We validate that, when not 0, FairSharing weight is greater than this quantity.

#### Which issue(s) this PR fixes:
Fixes #4247

#### Special notes for your reviewer:
We disallow the threshold value, as otherwise we have no way of validating that the value the user provided was the smallest possible value which can be represented accurately, or an even smaller value which collapsed to this value.
