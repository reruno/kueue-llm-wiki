#### What this PR does / why we need it:

As we found in https://github.com/kubernetes-sigs/kueue/issues/6288#issuecomment-3352666301, the `gh issue list` command is enforced with 30 30-item limit by default.
So, the older issues could not be shown by default.

Hence, the `--search "in:title"` option could help with the exact issue name by the `gh issue list` command.
