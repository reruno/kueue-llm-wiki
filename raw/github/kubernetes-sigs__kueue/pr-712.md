#### What this PR does / why we need it:
Currently, the kueue-controller can not apply patches to events since the kueue-manager-role doesn't have the permission for that operation.

So, the kueue-scheduler can not do that operations and output the below error in log:

```shell
kueue-controller-manager-75dff58fc-tmpjv manager E0422 19:46:44.767307       1 event.go:267] Server rejected event '&v1.Event{TypeMeta:v1.TypeMeta{Kind:"", APIVersion:""}, ObjectMeta:v1.ObjectMeta{Name:"mpijob-pi-1-ef222.17585964e31bcc1c", GenerateName:"", Namespace:"tz-test", SelfLink:"", UID:"", ResourceVersion:"23624747", Generation:0, CreationTimestamp:time.Date(1, time.January, 1, 0, 0, 0, 0, time.UTC), DeletionTimestamp:<nil>, DeletionGracePeriodSeconds:(*int64)(nil), Labels:map[string]string(nil), Annotations:map[string]string(nil), OwnerReferences:[]v1.OwnerReference(nil), Finalizers:[]string(nil), ManagedFields:[]v1.ManagedFieldsEntry(nil)}, InvolvedObject:v1.ObjectReference{Kind:"Workload", Namespace:"tz-test", Name:"mpijob-pi-1-ef222", UID:"4a015ae9-c2b1-45eb-953e-68b5a03cddc4", APIVersion:"kueue.x-k8s.io/v1beta1", ResourceVersion:"23625149", FieldPath:""}, Reason:"Pending", Message:"couldn't assign flavors to pod set worker: insufficient quota for memory in flavor default-flavor in ClusterQueue", Source:v1.EventSource{Component:"kueue-admission", Host:""}, FirstTimestamp:time.Date(2023, time.April, 22, 19, 45, 50, 0, time.Local), LastTimestamp:time.Date(2023, time.April, 22, 19, 46, 44, 766427229, time.Local), Count:9, Type:"Normal", EventTime:time.Date(1, time.January, 1, 0, 0, 0, 0, time.UTC), Series:(*v1.EventSeries)(nil), Action:"", Related:(*v1.ObjectReference)(nil), ReportingController:"", ReportingInstance:""}': 'events "mpijob-pi-1-ef222.17585964e31bcc1c" is forbidden: User "system:serviceaccount:kueue-system:kueue-controller-manager" cannot patch resource "events" in API group "" in the namespace "tz-test"' (will not retry!
```

https://github.com/kubernetes-sigs/kueue/blob/9ca57c86cf06c11a94a2d5b7badf60233a51a2f2/pkg/scheduler/scheduler.go#L418-L424

Therefore, I added a verb, `patch` to events.
