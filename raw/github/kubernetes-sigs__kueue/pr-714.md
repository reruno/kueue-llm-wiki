#### What this PR does / why we need it:
The `admittedWorkloadsPerQueue` uses the queue's name as a key only in `func AddClusterQueue`, even though it uses queueKey (Namespace/Name) as a key in other places for the cache.

Also, I changed the type of `admittedWorkloadsPerQueue` from `int` to `QUEUE_KEY: sets.Set[WORKLOAD_KEY]` to avoid being double counted workloads in `AddLocalQueue` and `AddClusterQueue`.

https://github.com/kubernetes-sigs/kueue/blob/9ca57c86cf06c11a94a2d5b7badf60233a51a2f2/pkg/cache/cache.go#L549-L560

https://github.com/kubernetes-sigs/kueue/blob/9ca57c86cf06c11a94a2d5b7badf60233a51a2f2/pkg/cache/cache.go#L420-L426
