/kind documentation

#### What this PR does / why we need it:

It's a KEP change to swap the default TAS Profile:
- promoting "Mixed" to default (*), and
- introducing a new feature gate (`BestFit`, deprecated from the start) to allow users to revert to the old behavior.

(*) However, the corresponding FeatureGate is formally kept deprecated. (See "notes" below).

#### Which issue(s) this PR fixes:

It's the first step to fix #7214 .

#### Special notes for your reviewer:

Keeping `TASProfileMixed` deprecated may feel strange but I did not find another way.

- Why not promote it to non-deprecated? Because there's a test preventing it.
- Why not remove it? Because some users may be still using it (quite likely so, given that it's our new default) - and then, removing it would make Kueue crash for them. \
  (I just checked; `kueue-controller-manager` fails to start if it's given an unrecognized FG name).
