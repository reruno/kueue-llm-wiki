<h2>Notable Changes</h2>
<ul>
<li>Fixed a <code>helm pull</code> regression from 3.18 - error pulling OCI charts with --password <a href="https://redirect.github.com/helm/helm/issues/31230">#31230</a></li>
<li>Fixed a <code>helm lint</code> regression from Helm 3.18 - rejected JSON Schema $ref URLs that worked in 3.17.x <a href="https://redirect.github.com/helm/helm/issues/31166">#31166</a></li>
<li>Fixed go mod tidy <a href="https://redirect.github.com/helm/helm/issues/31154">#31154</a></li>
<li>Fixed k8s version parsing not matching original <a href="https://redirect.github.com/helm/helm/issues/31091">#31091</a></li>
<li>Fixed charts failing when using a redirect registry <a href="https://redirect.github.com/helm/helm/issues/31087">#31087</a></li>
<li>Fixed missing debug logging for OCI transport</li>
<li>Fixed broken legacy docker support for login <a href="https://redirect.github.com/helm/helm/issues/30941">#30941</a></li>
<li>Fixed bugs from the move to ORAS v2</li>
<li>Fixed processing all hook deletions on failure <a href="https://redirect.github.com/helm/helm/issues/30673">#30673</a></li>
<li>Feature for <code>helm create</code> added httproute from gateway-api to create chart template <a href="https://redirect.github.com/helm/helm/issues/30658">#30658</a></li>
</ul>
