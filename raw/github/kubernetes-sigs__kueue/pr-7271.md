#### What this PR does / why we need it:
This PR removes the deprecated queue annotation, as preliminary cleanup for v1beta2. Users are expected to use the queue _label_ instead.

#### Which issue(s) this PR fixes:
Part of https://github.com/kubernetes-sigs/kueue/issues/7113

#### Special notes for your reviewer:

#### Does this PR introduce a user-facing change?
```release-note
Removed the deprecated workload annotation key "kueue.x-k8s.io/queue-name".

ACTION REQUIRED: Please ensure you are using the workload label "kueue.x-k8s.io/queue-name" instead.
```
