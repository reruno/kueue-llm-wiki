## Description

#### What type of PR is this?
/kind cleanup

#### What this PR does / why we need it:
As a part of the plan for v1beta2, LocalQueueFlavorStatus is marked as deprecated, and will be possibly dropped.

#### Which issue(s) this PR fixes:
Fixes #6777

#### Does this PR introduce a user-facing change?
```release-note
The .status.flavors in LocalQueue is deprecated, which will be removed in the future release.

ACTION REQUIRED:
You can consider migrating from the field usage to VisibilityOnDemand.
```

#### Notes:
Dropping LocalQueueFlavorStatus had been tested just to make sure deprecation is marked everywhere where applicable.
Dropping will be done as a part of a separate PR in the future once confirmed.
