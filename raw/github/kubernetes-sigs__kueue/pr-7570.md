#### What this PR does / why we need it:
This PR adds support for authentication to worker clusters using ClusterProfile API.

Changes:
* `MultiKueueCluster` API:
  * add `clusterProfile` field that references a `ClusterProfile` object that can be used to generate a rest config for the cluster
  * `kubeConfig` field is now optional, and validation ensures that either `kubeConfig` or `clusterProfile` is specified, but not both.
* `ClusterProfile` controller is added to watch for `ClusterProfile` objects and automatically creates and updates coresponding `MultiKueueCluster` objects.
* Config API:
  * add the `clusterProfiles` field to manage ClusterProfile API config, also add a feature gate for this feature.
