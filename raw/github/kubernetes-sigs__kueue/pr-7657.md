#### Special notes for your reviewer:

After running:

```
make kind-image-build test-e2e E2E_RUN_ONLY_ENV=true
```

we can now (in another terminal or in the same one if you chose N to skip cleanup) quickly rebuild and restart only the Kueue image with:

```
make kind-image-build test-e2e E2E_RUN_ONLY_KUEUE=true
```
