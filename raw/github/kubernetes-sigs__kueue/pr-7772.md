## Description

During upgrades with pre-existing LQ/CQ, the API server fails conversion with 'x509: certificate signed by unknown authority' because cert-controller doesn't inject CA bundles into CRD conversion webhook configurations. This fix manually injects CA bundles into kueue.x-k8s.io CRDs, registers webhooks before manager start so conversion endpoints are ready when caches sync.
