#### What this PR does / why we need it:

It simplifies the following pattern in test setup:

```golang
fwk.StartManager(ctx, cfg, func(ctx context.Context, mgr manager.Manager) {
    X(ctx, mgr)
})
```

into

```golang
fwk.StartManager(ctx, cfg, X)
```

(where `X` happens to be `managerSetup` in all cases).

Following up on [this review comment](https://github.com/kubernetes-sigs/kueue/pull/7794#discussion_r2549655978).
