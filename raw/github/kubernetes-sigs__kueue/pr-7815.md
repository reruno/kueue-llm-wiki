#### What this PR does / why we need it:
This PR adds several improvements to the code merged in https://github.com/kubernetes-sigs/kueue/pull/7312.

Three issues are addressed:
1. The patch is performed in strict mode, meaning it will fail if another process updated the pod status in the meantime.
2. It uses typed events in the event filter.
3. It introduces a new `podEligibleForTermination` helper to be used in the reconciler and event filters.
