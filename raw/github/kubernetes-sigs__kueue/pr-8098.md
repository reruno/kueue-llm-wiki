## Pull request overview

This PR removes outdated comments from the controller manager configuration file. The comments incorrectly suggested that users need to manually enable pod integration before enabling deployment, statefulset, or leaderworkerset frameworks. In reality, these frameworks automatically enable pod integration through the `ImplicitlyEnabledFrameworkNames` mechanism, making the comments misleading.

- Removed three misleading "# requires enabling pod integration" comments from the configuration
