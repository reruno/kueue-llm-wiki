#### What this PR does / why we need it:
A follow-up on the comment: https://github.com/kubernetes-sigs/kueue/issues/7742#issuecomment-3570619570
To skip Reconcile of MultiKueueCluster when ClusterProfile or Secret is NotFound.

Remove cache setup for ClusterProfile in main.go, as setting event handler does it automatically.
