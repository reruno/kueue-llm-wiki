#### What this PR does / why we need it:

This PR ensures that the RunWithPodSetsInfo function for TrainJobs is idempotent. It introduces a filtering mechanism to remove existing PodTemplateOverrides that were previously added by Kueue (identified by the PodSetLabel) before appending new ones.

Without this change, retrying an update operation could lead to duplicate overrides in the TrainJob specification.

Found during parallel execution of e2e tests.
