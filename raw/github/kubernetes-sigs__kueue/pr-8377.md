## Description

#### What type of PR is this?

/kind cleanup

#### What this PR does / why we need it:

This PR refactors the `util` package from the `cmd/kueuectl/app/util` directory into three packages:

- `flags` (`cmd/kueuectl/app/flags`)
- `dryrun` (`cmd/kueuectl/app/dryrun`)
- `clientgetter` (`cmd/kueuectl/app/clientgetter`)

#### Which issue(s) this PR fixes:

Updates #5948

#### Special notes for your reviewer:

This PR also includes a few renamings to avoid repetitions, e.g.:

- `clientgetter.NewClientGetter` -> `clientgetter.New`
- `dryrun.DryRunStrategy` -> `dryrun.Strategy`

Not all `util` packages are renamed in this PR. That's why I use "Updates" and not "Closes" before the issue number.

```release-note
NONE
```
