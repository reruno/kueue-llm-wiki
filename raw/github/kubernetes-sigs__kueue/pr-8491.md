#### What this PR does / why we need it:
The code works in place, while returning values to the caller as well. This introduces some unclarity whether this function works in-place or not.

This function started out with 1 returned value, then 2 and now it's 5. So at this point returning them is slightly confusing, unless I'm missing something.

Fixes #

#### Special notes for your reviewer:
Second PR for #8036.

I also changed lines like:
```go

```
to mins:
```go
min(child.state-child.stateWithLeader, minStateWithLeaderDifference)
```
