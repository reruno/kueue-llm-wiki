## Pull request overview

This PR refactors DRA (Dynamic Resource Allocation) related code by extracting a hardcoded driver name into a reusable constant from an external dependency. The changes follow up on feedback from PR #8421 to improve code maintainability.

Key changes:
- Added `sigs.k8s.io/dra-example-driver` v0.2.1 dependency to provide a shared constant for the DRA driver name
- Replaced all hardcoded "gpu.example.com" string literals in test code with `draconsts.DriverName`
- Updated version handling in Makefile-deps.mk to dynamically extract the version from go.mod instead of hardcoding it
