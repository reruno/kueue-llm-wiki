This PR is a manual cherry-pick of #8441 to the release-0.14 branch, implementing deduplication of cohort hierarchy processing to avoid redundant requeuing of inadmissible workloads when multiple ClusterQueues share the same cohort root.

**Key changes:**
- Added deduplication logic using a `processedRoots` set to track already-processed cohort roots
- Added comprehensive test coverage for the deduplication behavior and cohort cycle handling
