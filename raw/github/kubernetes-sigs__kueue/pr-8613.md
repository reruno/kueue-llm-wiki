## Description

Add tests to verify LeaderWorkerSet behavior with different restart policies:
- RecreateGroupOnPodRestart: all pods in the group are recreated when one is deleted
- NoneRestartPolicy: only the deleted pod is recreated, others remain unchanged

Also add RestartPolicy wrapper method to leaderworkerset test utilities.
