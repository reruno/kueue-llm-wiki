## Description

This PR fixes non-JSON structured logging in scheduler fair sharing.

Specifically:
- Replaces fmt.Sprintf-based string serialization of DRS values
- Uses structured fields to ensure valid JSON logs
- Aligns logging with controller-runtime / logr best practices
