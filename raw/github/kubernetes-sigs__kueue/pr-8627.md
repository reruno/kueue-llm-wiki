<h2>Notable Changes</h2>
<ul>
<li>Fixed bug where helm uninstall with --keep-history did not suspend previous deployed releases <a href="https://redirect.github.com/helm/helm/issues/12556">#12556</a></li>
<li>Fixed rollback error when a manifest is removed in a failed upgrade <a href="https://redirect.github.com/helm/helm/issues/13437">#13437</a></li>
<li>Fixed check to ensure CLI plugin does not load with the same name as an existing Helm command</li>
<li>Fixed helm test --logs failure with hook-delete-policy &quot;hook-failed&quot; or &quot;hook-succeed&quot; <a href="https://redirect.github.com/helm/helm/issues/9098">#9098</a></li>
<li>Fixed a bug where empty dependency lists were incorrectly treated as present</li>
<li>Fixed a bug where the watch library did not only watch namespaces associated with the objects</li>
<li>Fixed regression in downloader plugins environment variables <a href="https://redirect.github.com/helm/helm/issues/31612">#31612</a></li>
<li>Fixed bug where --server-side flag is not respected with helm upgrade --install <a href="https://redirect.github.com/helm/helm/issues/31627">#31627</a></li>
<li>For SDK users: exposed KUBECONFIG to env</li>
</ul>
