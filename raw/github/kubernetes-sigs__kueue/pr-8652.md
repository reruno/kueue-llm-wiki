#### What this PR does / why we need it:

This PR eliminates usage of the `Evicted()` helper in unit tests and replaces it with `EvictedAt(time.Time)`.

As suggested by @mimowo, an `EvictedAt` helper is added to `WorkloadWrapper`, mirroring existing helpers such as `AdmittedAt` and `ReserveQuotaAt`. The helper explicitly sets the `WorkloadEvicted` status condition with a deterministic `LastTransitionTime`.


#### Which issue(s) this PR fixes:

Fixes #8633

#### Special notes for your reviewer:

- `EvictedAt` sets the eviction condition with an explicit `LastTransitionTime`
- All unit tests previously using `Evicted()` have been migrated
- No production code behavior is changed

```
NONE
```
