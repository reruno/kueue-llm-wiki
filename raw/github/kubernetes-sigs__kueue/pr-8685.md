#### What this PR does / why we need it:
Alongside the https://github.com/kubernetes-sigs/kueue/pull/8618 approach, I added the `kueue.x-k8s.io/pod-index-offset` annotation mechanism, which allows us to manage Pod Index offset even if Cross-PodSet scheduling (`kueue.x-k8s.io/podset-group-name`) is disabled.
