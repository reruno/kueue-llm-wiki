<h2>Notable Changes</h2>
<ul>
<li>Feature: added chart name to dependency logs, namespace to resource waiting logs, and confirmation message when all resources are ready <a href="https://redirect.github.com/helm/helm/issues/31530">#31530</a></li>
<li>Feature: improved plugin name validation error messages and field name detection <a href="https://redirect.github.com/helm/helm/issues/31491">#31491</a></li>
<li>Feature: improved the --wait flag by allowing explicit strategy selection (including explicit --wait=hookOnly) and preventing SDK timeout errors when timeout is not specified <a href="https://redirect.github.com/helm/helm/issues/31421">#31421</a></li>
<li>Feature: allow concurrent dependency build with atomic file write <a href="https://redirect.github.com/helm/helm/issues/30984">#30984</a></li>
<li>Feature: added a --no-headers flag to the 'helm repo list' command, allowing users to suppress table headers in the output. Useful for scripting and automation <a href="https://redirect.github.com/helm/helm/issues/31448">#31448</a></li>
<li>SDK feature: added a LoadArchive to common loader <a href="https://redirect.github.com/helm/helm/issues/31462">#31462</a></li>
<li>SDK feature: introduced support for custom kstatus readers <a href="https://redirect.github.com/helm/helm/issues/31706">#31706</a></li>
<li>Fixed bug where a plugin name could already be used by another command <a href="https://redirect.github.com/helm/helm/issues/31427">#31427</a></li>
<li>Fixed bug where --server-side flag was not passed to install when using upgrade --install <a href="https://redirect.github.com/helm/helm/issues/31635">#31635</a></li>
<li>Fixed bug where HELM_ environment variables were not passed to plugins. this fixes a regression which was blocking some getter plugins <a href="https://redirect.github.com/helm/helm/issues/31613">#31613</a></li>
<li>Fixed bug where Helm test --logs failed with hook-delete-policy &quot;hook-failed&quot; or &quot;hook-succeed&quot; <a href="https://redirect.github.com/helm/helm/issues/31579">#31579</a></li>
<li>Fixed kube client logging issue <a href="https://redirect.github.com/helm/helm/issues/31560">#31560</a></li>
<li>Fixed regression where vendor-specific suffixes were stripped from .Capabilities.KubeVersion.GitVersion, breaking charts that detect managed Kubernetes platforms <a href="https://redirect.github.com/helm/helm/issues/31528">#31528</a></li>
<li>Fixed a bug where helm uninstall with --keep-history did not suspend previous deployed releases <a href="https://redirect.github.com/helm/helm/issues/12564">#12564</a></li>
<li>SDK: bump k8s API versions to v0.35.0</li>
<li>docs: updated <code>helm template</code> help text to document --api-versions <a href="https://redirect.github.com/helm/helm/issues/31683">#31683</a></li>
<li>docs: fixed documentation about default wait strategy</li>
</ul>
