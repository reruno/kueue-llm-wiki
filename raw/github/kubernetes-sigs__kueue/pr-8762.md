#### What this PR does / why we need it:
The end-to-end performance test setup drifted. I pinpointed two changes that make it work, i.e. the scenarios are being run and the metric report is generated (and the test reports a success).

1. The `kueue.x-k8s.io/queue-name` is now a label, it used to be an [annotation](https://github.com/kubernetes-sigs/kueue/blob/6c0532e5fc9dbfa5dde437dcca0a923348153c44/pkg/constants/constants.go#L25).
2. The performance test directory was nested differently. Now the nesting is deeper - it went from `test/performance` to `test/performance/e2e/jobs`.
