## Description

**What type of PR is this?**
/kind cleanup

**What this PR does / why we need it:**
This PR adds a website link verification tool and integrates it into the existing make verify  workflow..

It adds a target `make-verify-website-links` to Makefile-verify.mk that runs `linkchecker` on https://kueue.sigs.k8s.io/ and identifies broken links

**Which issue(s) this PR fixes:**
Fixes https://github.com/kubernetes-sigs/kueue/issues/8164.
