## Pull request overview

This PR fixes a bug where finished workloads prevented ClusterQueue deletion by blocking finalizer removal. The fix makes the scheduler cache's update path consistent with its rebuild path by also checking `IsFinished()` when processing workload updates.

**Changes:**
- Modified the cache update logic to exclude finished workloads from the scheduler cache
- Added unit test to verify finished workloads are removed from cache during updates
- Added integration test to verify ClusterQueue finalizer removal works correctly with finished workloads
