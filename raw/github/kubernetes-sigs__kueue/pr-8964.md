## Description

#### What type of PR is this?
/kind feature

#### What this PR does / why we need it:

This PR introduces the `TASReplaceNodeOnNodeTaints` feature gate, which enables Kueue's Topology Aware Scheduling to handle nodes tainted with `NoExecute`.

Previously, TAS might not have reacted appropriately when a node became unhealthy due to taints, potentially leaving workloads in a broken state.
