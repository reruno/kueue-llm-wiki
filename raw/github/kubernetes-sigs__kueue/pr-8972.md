## Description

The ElasticJob multikueue integration test was failing with `Index with name field:metadata.workloadSliceName does not exist`. The WorkloadSliceNameKey Pod index is only registered during indexer.Setup() if `ElasticJobsViaWorkloadSlices` feature gate is enabled at that time.
