## Description

This PR implements the initial steps outlined in the [quota release strategy KEP](https://github.com/kubernetes-sigs/kueue/tree/main/keps/6143-quota-release-strategy). It adds a feature flag specific to the pod integration that will cause pods to be considered inactive in the same way that other integrations do (when termination is initiated, not completed)

/kind feature
