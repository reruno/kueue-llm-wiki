## Pull request overview

This PR refactors resource comparison utilities by moving the `GetGreaterKeys` function from `pkg/util/resource` to the `pkg/resources` package as a method on the `Requests` type. It also renames `NewRequestsFromPodTemplate` to `NewRequestsFromPodSpec` for better consistency and adds new helper methods.

**Changes:**
- Renamed `NewRequestsFromPodTemplate` to `NewRequestsFromPodSpec` and simplified it to accept `PodSpec` instead of `PodTemplateSpec`
- Moved `GetGreaterKeys` functionality from `pkg/util/resource` to `pkg/resources.Requests.GreaterKeys` method
- Added `GreaterKeysRL` helper method to compare Requests against a ResourceList
- Added `DropPrefixes` method for filtering resources by prefix (currently unused)
- Updated all callers across the codebase to use the new APIs
