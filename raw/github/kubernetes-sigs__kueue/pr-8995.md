## Pull request overview

This PR refactors the `NewRequestsFromPodTemplate` function to accept a pointer to `PodTemplateSpec` instead of passing it by value. This improves performance by avoiding unnecessary copying of the relatively large `PodTemplateSpec` struct and aligns with established patterns in the codebase where `PodTemplateSpec` is consistently passed by pointer.

**Changes:**
- Modified `NewRequestsFromPodTemplate` function signature to accept `*corev1.PodTemplateSpec` instead of `corev1.PodTemplateSpec`
- Updated the call site in `podSetTopologyRequest` to pass a pointer to the template
