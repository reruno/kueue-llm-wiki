## Pull request overview

Removes a temporary Go sumdb/proxy workaround from the `yq` download/install target now that the repository pins `github.com/mikefarah/yq/v4` at a fixed version that no longer triggers the upstream zip filename issue.

**Changes:**
- Deletes the `GONOSUMDB` / `GOPROXY=direct` environment overrides for installing `yq`.
- Simplifies the `yq` make target to match the standard `go install ...@$(VERSION)` pattern used by other tools.
