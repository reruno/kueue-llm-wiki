## Pull request overview

This PR fixes a subtle bug in the workload comparison function where comparing a sticky workload with itself would incorrectly return a non-zero result, violating the invariant that `lessFunc(a,a)` should always be false.

**Changes:**
- Refactored the sticky workload comparison logic to check if both workloads have different sticky states before prioritizing one over the other
