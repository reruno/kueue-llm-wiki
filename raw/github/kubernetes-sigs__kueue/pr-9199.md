#### What this PR does / why we need it:
Revive v1.14.0 (included in golangci-lint v2.9.0) adds a new var-naming check that flags package names conflicting with Go standard library names. Suppress it since these are established package names that cannot be reasonably renamed.

First commit updates golangci to v2.9.0: https://github.com/kubernetes-sigs/kueue/pull/9196
