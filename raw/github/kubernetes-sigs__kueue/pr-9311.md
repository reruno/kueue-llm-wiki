#### What this PR does / why we need it:

This change adds configurable security contexts for KueueViz backend and frontend deployments, enabling users to run KueueViz in clusters with restricted pod security profiles.

The implementation mirrors the existing controller manager security context configuration with secure defaults:
- runAsNonRoot: true
- seccompProfile.type: RuntimeDefault
- readOnlyRootFilesystem: true
- allowPrivilegeEscalation: false
- capabilities.drop: ALL
