### Comment by [@IrvingMg](https://github.com/IrvingMg) — 2026-02-19T16:45:03Z

Verification steps:

1. Uncommented the Prometheus resource in `config/default/kustomization.yaml` (`- ../components/prometheus`)
2. Uncommented the patches block in `config/components/prometheus/kustomization.yaml`:
```yaml
patches:
- path: monitor_tls_patch.yaml
  target:
    kind: ServiceMonitor
```
3. Built and checked the output `./bin/kustomize build test/e2e/config/certmanager/ | grep serverName` and got the expected output: 
```yaml
serverName: kueue-controller-manager-metrics-service.kueue-system.svc
```
before was `serverName: kueue-webhook-service.kueue-system.svc`.

So this is working. Thanks!
