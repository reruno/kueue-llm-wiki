#### What this PR does / why we need it:
During [Multikueue dev setup](https://kueue.sigs.k8s.io/docs/tasks/dev/setup_multikueue_development_environment/#setup-multikueue-with-tas) process, a ServiceAccount is created for each of worker clusters. SA token expires in 24h, leading to need for a restart of the entire MK dev setup. To avoid this, increase expiration time to 7d.

This PR creates long-live API token for SA token during MultiKueue development setup. This approach is used in non-dev [MultiKueue setup docs](https://kueue.sigs.k8s.io/docs/tasks/manage/setup_multikueue/).
