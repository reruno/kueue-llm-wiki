#### Special notes for your reviewer:

The issue was that the script was spitting by @:./, then counts forward. However this is not working well on repos like `k8s.io`. Instead, it is better to split just by "/" and count backwards. Using "NF" number of fields to count backwards. 

Also extracted this logic to helpers.
