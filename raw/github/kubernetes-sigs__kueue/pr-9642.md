#### What this PR does / why we need it:

Updates KEP `kep.yaml` files that were not updated when their associated feature gates were promoted to GA in v0.17:

- **KEP 2936 (LocalQueueDefaulting)**: `stage: alpha` → `stage: stable`
- **KEP 1618 (ObjectRetentionPolicies)**: Updated stage, latest-milestone, milestone, and disable-supported
- **KEP 693 (MultiKueue)**: Updated latest-milestone, milestone, and disable-supported for MultiKueueBatchJobWithManagedBy GA
- **KEP 1834 (PropagateBatchJobLabelsToWorkload)**: Updated stage, latest-milestone, and milestone
