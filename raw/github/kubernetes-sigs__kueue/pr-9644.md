#### What this PR does / why we need it:
This PR backports the test/feature-gate cleanup from #9600 to release-0.16 with branch-specific adjustments to resolve conflict ([source](https://github.com/kubernetes-sigs/kueue/pull/9600#issuecomment-3985872164)).

Branch-specific adjustments
- Added explicit TAS feature-gate setup in tas_test.go for branch-stable behavior:
  - TASFailedNodeReplacementFailFast
  - TASReplaceNodeOnNodeTaints
  - TASReplaceNodeOnPodTermination (per test scenario)

Not backported intentionally
- `test/integration/multikueue/dra_test.go` and related additions were not included.
  - Reason: these changes are outside the intended scope for this backport PR and are not required for the target branch fix.

Why this differs from main/0.17
- Feature-gate defaults and test assumptions differ across branches.
- This backport preserves release-0.16 semantics rather than mechanically copying the main-branch diff.
