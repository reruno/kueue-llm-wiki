#### What this PR does / why we need it:
This PR backports the test/feature-gate cleanup from #9600 to release-0.15 with branch-specific adjustments to resolve conflict ([source](https://github.com/kubernetes-sigs/kueue/pull/9600#issuecomment-3985868555)).

**File not backported in 0.15:**
- `test/integration/multikueue/dra_test.go` and related new test dependencies were **not** included.
  - Reason: some files/helpers (e.g. `util.CreateAdmissionChecksAndWaitForActive()`) are not part of the release-0.15 baseline and would require pulling additional test scaffolding (non-minimal change).

Difference vs main/0.17:
- Some gates/defaults differ across branches. Therefore this backport preserves semantics for release-0.15 instead of mechanically matching main diff.
