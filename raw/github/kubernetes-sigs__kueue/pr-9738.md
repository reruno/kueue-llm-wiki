#### What this PR does / why we need it:

The preemption pruning logic in ordering.go uses `drs.IsZero()` as a proxy for "not currently borrowing." This conflates the numerical DRS ratio with the real-time borrowing state. With historical-borrowing-based DRS (#9737), the ratio may be non-zero even when a node is not actively borrowing, breaking this assumption.

This PR adds an explicit `IsBorrowing()` method to DRS, set from the existing borrowing computation in `dominantResourceShare()`. The two pruning sites in ordering.go now use `!drs.IsBorrowing()` instead of `drs.IsZero()`.

No behavioral change with the current DRS computation.
