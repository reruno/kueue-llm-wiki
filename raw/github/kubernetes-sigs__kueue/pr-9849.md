<h2>Notable Changes</h2>
<ul>
<li>Fixed a bug where --dry-run=server was not respecting generateName <a href="https://redirect.github.com/helm/helm/issues/31563">#31563</a></li>
<li>Fixed a bug where empty CRD resources caused a crash. Now it prints an error <a href="https://redirect.github.com/helm/helm/issues/31578">#31578</a></li>
<li>Fixed a bug where OCI references with tag+digest failed with &quot;invalid byte&quot; error <a href="https://redirect.github.com/helm/helm/issues/31601">#31601</a></li>
<li>Fixed a bug where user-provided nil value was not preserved when chart has an empty map or no default for a key <a href="https://redirect.github.com/helm/helm/issues/31644">#31644</a></li>
<li>Fixed a regression since Helm 3.18.0 where Pulling charts from OCI repositories that use an index to store both Container Images and Helm Charts under the same tag failed <a href="https://redirect.github.com/helm/helm/issues/31776">#31776</a></li>
<li>Fixed a Helm 4 regression where gotemplate white space trimming directly after YAML doc separators, when present after postrendering, caused YAML file corruption <a href="https://redirect.github.com/helm/helm/issues/31868">#31868</a></li>
<li>Fixed a bug where <code>FailedStatus</code> is treated as a terminal state, causing upgrades to fail prematurely when cluster autoscalers needed time to provision nodes, or when pods were being deleted during rolling updates <a href="https://redirect.github.com/helm/helm/issues/31897">#31897</a></li>
<li>Fixed broken backwards compatibility for deprecated <code>--atomic</code> flag on install command <a href="https://redirect.github.com/helm/helm/issues/31901">#31901</a></li>
<li>SDK: Fixed a Windows 'Access Deined' error if multiple processes try to download the same chart version concurrently <a href="https://redirect.github.com/helm/helm/issues/31128">#31128</a></li>
<li>SDK: Fixed a bug where users did not receive any logs from the waiter, causing confusion as several minutes could pass with no user feedback <a href="https://redirect.github.com/helm/helm/issues/31717">#31717</a></li>
<li>SDK: Fixed a bug where server-side apply defaults did not always match the CLI defaults <a href="https://redirect.github.com/helm/helm/issues/31732">#31732</a></li>
<li>SDK: Fixed a Go import issue when downstream tooling attempted to vendor helm.sh/helm/v4/pkg/kube <a href="https://redirect.github.com/helm/helm/issues/31852">#31852</a></li>
</ul>
